# Smart Navigation Aid for Visually Impaired Users

## Project Overview
This project focuses on developing a smart navigation assistance system for visually impaired individuals using deep learning-based object detection. The system detects obstacles such as people, vehicles, stairs, traffic signs, and barriers in real time.

## Technologies Used
- Python
- PyTorch
- YOLOv8 (Ultralytics)
- OpenCV
- NumPy
- Google Colab (GPU Training)

## Repository Structure
- src: Source code for training and detection
- dataset: Custom annotated dataset in YOLO format
- models: Trained model weights
- notebooks: Google Colab training notebook
- results: Detection outputs and screenshots
- docs: Reports and experiment logs
